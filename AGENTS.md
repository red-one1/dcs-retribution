# Agent Instructions

This file ensures agents keep memory across sessions.

## Required Startup Steps
1. Read AGENT_MEMORY.md before doing any work.
2. If it is missing, create it using the template sections: Project Overview, Decisions, Open Questions, TODO, Session Recap.

## Required Shutdown Steps
1. Append a short entry to the top of the Session Recap in AGENT_MEMORY.md.
2. Note any new decisions, open questions, or TODOs.
